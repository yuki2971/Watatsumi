
Watatsumi

海を資源に変える、潮流と塩の工業革命。
A NeoForge 1.21.1 mod focused on Oceanic Industrial Expansion.

🌊 コンセプト

Watatsumiは、海を中心に発展する海洋工業拡張MODです。
ネーミングは日本神話の海神　"ワタツミ"　から(開発者が日本人なので)
余りがちな素材や海バイオームを活かし、潮流・塩・海水を資源へと変換します。

海依存の発電と精製

余り素材の再定義

段階的に拡張される技術ツリー

⚙️ コアデザイン

Ocean-Driven
海水・潮流・海バイオームを前提とした機械。

Resource Revaluation
丸石・砂・銅・昆布などの“余り素材”に新しい役割を。

Tiered Progression
潮力 → 電解 → 精製 → 海底プラントへ。

なぜWatatsumiは海洋なのか？

Create中心環境で絶海の孤島からスタートしたとき、
海は広いのに“やること”が少なかった。

プリズマリンは作れる。
だが使い道がない。

海は資源の宝庫であるはずなのに、
工業の主役になれない。

だから海を「工業の起点」にする。
