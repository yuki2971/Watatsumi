# Watatsumi Dev Log — Wooden Tub 実装

## 概要

Watatsumiの初期装置として **Wooden Tub（木桶）ブロック**を実装。
将来的には **海水 → 塩生成装置**として使用予定。

---

# 実装内容

## 1. ブロック登録

`ModBlocks` に Wooden Tub を登録。

```java
public static final DeferredHolder<Block, Block> WOODEN_TUB =
        BLOCKS.register("wooden_tub",
                () -> new WoodenTubBlock());
```

---

## 2. WoodenTubBlock クラス

ブロックの基本設定

```java
public class WoodenTubBlock extends Block {

    public WoodenTubBlock() {
        super(BlockBehaviour.Properties.of()
                .strength(1.5F)
                .sound(SoundType.WOOD)
                .noOcclusion());
    }
}
```

設定内容

* 強度: `1.5`
* 音: `SoundType.WOOD`
* 空洞ブロックのため `noOcclusion()`

---

## 3. カスタムモデル

桶は **空洞構造**のため
`cube` モデルではなく **elements構造のJSON**を使用。

モデル構造

```
底
+ 4つの壁
= 中央が空洞
```

高さ

```
ブロック高さ: 12
```

## 6. VoxelShape（当たり判定）

見た目が空洞のため
デフォルトのフルブロック当たり判定を修正。

```java
private static final VoxelShape SHAPE = Shapes.or(
        Block.box(0,0,0,16,2,16),
        Block.box(0,2,0,2,12,16),
        Block.box(14,2,0,16,12,16),
        Block.box(2,2,0,14,12,2),
        Block.box(2,2,14,14,12,16)
);
```

```java
@Override
public VoxelShape getShape(BlockState state, BlockGetter level, BlockPos pos, CollisionContext context) {
    return SHAPE;
}
```

これにより

```
中央が空洞
プレイヤーが桶の中に入れる
```

挙動になる。
